高二：紅綠燈視覺辨識專案

在高中時就因為注意到人工智慧越來越夯，因此我開始主動在課餘時間做了一些關於 AI 的專題和研究。其中最讓我印象深刻的是紅綠燈辨識專案，我訓練出來的辨識模型最後達到了90%以上的準確率。

起初，我有先看政大蔡炎龍教授在政大 moocs 上的人工智慧的相關課程。在當時就做了簡單手寫數字辨識，這是我第一次親眼見證原來電腦也可以像人類一樣做到辨識影像的功能，這讓我感到相當的新穎。這讓我想要挑戰對於當時的我較為困難的紅綠燈辨識專案。

這個專案主要是由YOLOv4這個物件辨識的模組來完成，主要是因為當時我看到了YOLOv4的論文上寫YOLOv4可以做到很快速的即時偵測、訓練時的硬體資源不需耗費太大、準確率很高，因此我採用了YOLOv4來實作。而在做這個專案中我遇到以下幾個難題：

  1. 網路上的資料庫幾乎沒有台灣紅綠燈的照片，因此為了更加貼近台灣當地的情形，我就在街道上拍了3000張以上的照片，並且一一標註 label，建立起自己的資料庫。
  2. 在當時我能找到的標註影像的軟體都無法將標註結果輸出成我需要的 YOLO 格式，因此我就先以預設格是輸出，再轉為我需要的 YOLO 格式。
  3. YOLOv4可以在我完成訓練後自動畫出 learning curve，但是因為訓練時間費時好幾天，過程中無可避免地會中斷訓練，因此自動畫出來的 learning curve 不完整，因此當時我收集訓練中產生的 log 檔，從中挑選出我需要的資料後再以 matploblib 畫出 learning curve（如圖1）。此外，我也有畫出confusion matrix 幫助我分析模型（如圖2）。

這個辨識模型可以辨識輸入的紅綠燈照片，模型會辨識出燈號，並且將紅綠燈的位置框出來。(如圖3)

這個專案的成功不僅讓我對於寫程式產生了信心，同時更進一步讓我想要研究更多更專業深入的機器學習相關技術，像是我後來主動閱讀台大資工的徐宏民教授指導學生的論文 xCos:An Explainable Cosine Metric for Face Verification Task來閱讀，從這篇論文中讓我認識到了可解釋性人工智慧，這可以讓使用者了解模型如何進行判斷。我也有主動閱讀台大電機的李宏毅教授指導學生的論文Completely Unsupervised Phoneme Recognition by Adversarially Learning Mapping Relationships from Audio Embeddings，從這篇論文我體認到了半監督式學習以及非監督式學習的強大。這個專案也讓我在後來都抱持著「我一定可以克服的」心態來面對往後的挑戰。
